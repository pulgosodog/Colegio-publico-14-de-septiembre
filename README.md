# Colegio Publico 14 de Septiembre — Website

A small, static website for Colegio Público 14 de Septiembre (Managua, Nicaragua). This project provides an informational landing page for the school with sections for About, Modalities, Contact, and an embedded map. It uses plain HTML, CSS and small vanilla JavaScript widgets.

## Features

- **Informational pages**: Hero, About, Modalities, Apply, Contact, and Footer.
- **Accessible carousel**: Lightweight JS carousel for the About section (`main.js`).
- **Smooth scrolling**: Smooth in-page navigation for anchors (`smooth-scroll.js`).
- **Embedded map**: Google Maps iframe showing the school's location.

## Technologies

- HTML5 (`index.html`)
- CSS (`styles.css`) — uses Inter font from Google Fonts
- JavaScript (`main.js`, `smooth-scroll.js`) — no frameworks

## How to view

The site is static. To view it locally, open `index.html` in your browser or serve the folder with a simple HTTP server. Example using any static server or VS Code Live Server extension.

The site is also published on GitHub Pages at: https://pulgosodog.github.io/Colegio-publico-14-de-septiembre/

## Project structure

- `index.html` — main page and layout
- `styles.css` — styling and layout rules
- `main.js` — carousel behavior for the About section
- `smooth-scroll.js` — smooth in-page anchor scrolling
- `assets/` — images and icons used by the site (logo, photos, favicon)

Notable files in `assets/`: `logo.png`, `Matricula_2025.jpg`, `car_1.jpg`..`car_5.jpg`, `facebook.png`, `favicon.ico`, `Preescolar.png`, `primaria.jpg`, `secundaria.jpg`, `salon.jpg`, `adultos.png`.
